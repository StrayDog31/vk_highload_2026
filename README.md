# Аналог Tinder

## 1. Тема и целевая аудитория
### 1.1 Тема
Tinder — международное геосоциальное приложение для знакомств, которое позволяет:

- Находить потенциальных партнеров для общения и романтических отношений на основе взаимного интереса
- Использовать механизм свайпов для быстрой оценки профилей
- Обмениваться сообщениями в чате после образования мэтча
- Использовать рекомендательную систему на основе поведения пользователя и машинного обучения

География и покрытие — всемирно.

### 1.2 Функционал MVP
MVP включает в себя:

- Регистрация и авторизация
- Редактирование анкеты
- Карточка анкеты (отображение фото, имени, возраста и краткой информации о пользователе)
- Механика свайпов
- Рекомендательная система
- Чат

### 1.3 Целевая аудитория

MAU — 75 млн. человек
DAU — 35 млн. человек (весьма разыне данные на разынх ресурсах)

Основные регионы:
- США
- Латинская Америка (Бразилия, Аргентина)
- Европа

Средний возраст: 18–35 лет (ядро аудитории — миллениалы и поколение Z)

Гендерное распределение:
- Женщины: 75%
- Мужчины: 25%
